## BOM
It will use : 
- 1x Seeed XIAO RP2040
- 1x 0.91 inch OLED display
- 12x 1N4148 Diodes
- 12x MX style switch
- 12x Blank DSA keycaps 
- 12x SK6812 MINI-E
- 2x 3d printed parts
- 3x M3x16mm screw
- 3x M3x5mx4mm heatset insert
OPTIONNAL
- 2x 6x6 spst push button i have some laying around so id figured might aswell use them :D
- 1x Li-ion battery that fits inside, if you want to use ble and zmk firmware. There is a pin header for this case. 
**DISCLAMER!! BE CAREFUL USING LI-ION BATTERY. I WILL NOT BE RESPONSIBLE FOR ANY DAMAGES THAT MAY OUTCOME FROM THIS HARDWARE**
